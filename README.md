FinanceTracker — Full-stack приложение для управления личными финансами, отслеживания доходов/расходов и контроля баланса на счетах.

**Backend**	- .NET 8, ASP.NET Core Web API, Entity Framework Core
**Database** - SQLite
**Frontend** - Vue.js 3, Axios, Vite
**Testing**	- xUnit, Moq, FluentAssertions
